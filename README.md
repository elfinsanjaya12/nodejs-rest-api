# Demo NodeJS REST Api

## Requirement
For running this project please install nodejs and mongodb.
NodeJS:
```
node --version
v10.16.0
```
MongoDB:
```
mongo --version
MongoDB shell version v4.2.0
```

## Getting Started
Install npm package:
```
npm install
```

Copy configuration file
```
cp .env.example .env
```

Run the server:
```
npm run start
```

## Run The Test
Copy configuration file
```
cp .env.example .env.test
```

Run test
```
npm run test
```
